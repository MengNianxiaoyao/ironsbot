# 帮助插件

## 查看插件列表

发送 `帮助`，机器人会回复带序号的插件列表：

```
📖 插件列表：
1. 帮助 — 获取插件帮助信息
2. 赛尔号信息查询 — 赛尔号信息查询插件
3. 表情包 — 随机或是指定输出一张可爱表情包
...

💬 输入序号查看详细帮助 · 输入 0 退出
```

## 查看插件详细帮助

在列表显示后，直接输入**序号**即可查看对应插件的 `usage` 详情：

```
用户：帮助
机器人：📖 插件列表：（如上）

用户：2
机器人：📖 赛尔号信息查询（详细 usage 内容）

用户：3
机器人：📖 表情包（详细 usage 内容）

用户：0
机器人：❌ 已退出帮助
```

- 查看完一个插件后**无需重新发送 `帮助`**，可继续输入其他序号。
- 输入 `0` 退出选择；发送任何非数字命令会自动结束帮助会话。
